## 다운로드 — 아래 Assets에서 이것만 받으면 됩니다

- 🪟 **Windows** → `PostureFairy-Setup-{{VERSION}}.exe`
- 🍎 **macOS** (Intel / Apple Silicon 공용) → `PostureFairy-{{VERSION}}-universal.dmg`

(`Source code`나 `.blockmap`, `latest*.yml` 같은 나머지 파일은 자동 업데이트용/GitHub가 자동으로 붙이는 파일이라 무시하셔도 됩니다.)

## 설치 및 사용법

**Windows**
1. `PostureFairy-Setup-{{VERSION}}.exe` 실행 → 설치 진행
2. "Windows가 PC를 보호했습니다" 경고가 뜨면 **추가 정보 → 실행**
3. 카메라 권한 허용
4. 저장된 프로필이 없으면 자동으로 캘리브레이션 창이 뜹니다 — 안내에 따라 3단계 진행

**macOS**
1. `PostureFairy-{{VERSION}}-universal.dmg` 더블클릭 → 앱을 Applications 폴더로 드래그
2. Applications에서 **우클릭(또는 Control+클릭) → 열기** (그냥 더블클릭하면 "확인되지 않은 개발자" 경고로 막힙니다 — 서명되지 않은 앱이라 최초 1회만 이렇게 해주시면 됩니다)
3. 카메라 권한 허용
4. 저장된 프로필이 없으면 자동으로 캘리브레이션 창이 뜹니다 — 안내에 따라 3단계 진행

## 이후 사용

- 컴퓨터를 켤 때마다 자동으로 백그라운드 실행됩니다 (Windows 트레이 / macOS 메뉴바 아이콘으로 확인 가능)
- 나쁜 자세가 몇 초 유지되면 화면 우상단에 요정이 나타났다가 자동으로 사라집니다
- 다시 캘리브레이션하려면: 트레이/메뉴바 아이콘 우클릭 → "캘리브레이션 시작"
- 카메라가 이상하게 나오면: 다른 화상회의 앱이나 브라우저 탭이 카메라를 이미 쓰고 있지 않은지 확인해주세요 (웹캠은 동시에 한 프로그램만 접근 가능합니다)
- **Windows는 이후 업데이트가 자동으로 적용됩니다** — 백그라운드로 새 버전을 받아뒀다가 앱이 다음에 재시작될 때 조용히 반영되므로, 새 릴리즈가 나와도 다시 내려받아 재설치할 필요 없습니다. (macOS는 아직 자동 업데이트가 지원되지 않아 새 버전이 나오면 위 dmg를 다시 받아 설치해주세요.)

## 알려진 제한사항

- 코드 서명이 안 된 상태라 각 OS에서 "확인되지 않은 개발자" 경고가 뜹니다 (정상입니다, 위 안내대로 진행하시면 됩니다)
